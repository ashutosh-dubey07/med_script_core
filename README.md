# Med-Script Core

A secure, offline-first Prescription Audit System.

## Project Scope
- Reads messy medical prescriptions.
- Identifies medicine names and their core salts.
- Cross-checks them against a secure local database to flag banned or unsafe medicines.

## Core Setup
- **Language:** Python
- **Database:** SQLite (Local Vault)
- **Security:** Strict parameterized queries to prevent SQL Injection.
